Toggle Class

Since Bootstrap uses classes to decide whether or not something is visible/active, this can be a great way to use alternative interfaces to switch active tabs (e.g. like an image map).

Use the buttons below to switch what tab content is loaded. Notice in the HTML that we're only using the tab-content part of the tabbed interface example since we don't need the actualy tabs themselves anymore to handle navigation.

FYI, my implementation appears to kill the fade animation, so perhaps if you have the time, you can research a better way to do this that retains the fade!
