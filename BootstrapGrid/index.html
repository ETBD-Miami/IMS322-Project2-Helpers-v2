A Quick Overview of the Boostrap Grid System

The examples below are all built using the default Bootstrap styles, which are applied by using the predefined classes (and sometimes multiple classes per div). The associated style.css file is primarily only used to provide background color to each section to clarify the layout. Also, the layout is automatically responsive, with built-in default media query changes.

Container

Each main section should be inside an element with container class, which will automatically center and horizontally pad your siteâ€™s contents.
Row
A row is a wrapper for columns. Boostrap uses a 12 column system you can think of the page as being constructed from 12 columns, and an element can be anywhere from 1-12 columns wide.

"In a grid layout, content must be placed within columns and only columns may be immediate children of rows."

Notice that this text inside a row has no default padding, only container and column classes do. It's probably best practice not to put any actual content in the row element itself, only in the embedded column elements.
